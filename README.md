# Brave

Brave is a privacy-first browser company offering developer APIs for anonymous web search and privacy-preserving advertising. The Brave Search API provides real-time search results from an independent index of over 30 billion pages, designed for AI and LLM applications. The Brave Ads API enables advertisers to manage campaigns and access performance reporting for privacy-respecting ads.

## APIs

### Brave Search API
- **Base URL:** `https://api.search.brave.com/res/v1/`
- **Authentication:** API key via `X-Subscription-Token` header
- **Documentation:** https://api-dashboard.search.brave.com/app/documentation/web-search/get-started
- **Pricing:** $5/1,000 requests (Search), $4/1,000 requests + $5/M tokens (Answers)
- **Rate Limits:** 50 QPS (Search), 2 QPS (Answers)

### Brave Ads API
- **Base URL:** `https://ads-serve.brave.com`
- **Authentication:** API key via `X-Brave-Api-Key` header
- **Documentation:** https://ads-help.brave.com/campaign-performance/API/
